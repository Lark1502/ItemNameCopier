# ItemNameCopier

ItemNameCopier(物品名称复制)：魔兽世界时光服专用，可配合其他拍卖行插件使用。时光服拍卖行中使用Shift+点击填充搜索栏时，因为一些插件对物品链接的污染，会使拍卖行报错，直接输入或者复制粘贴物品名则不会。本插件在拍卖行打开时拦截 Shift/Ctrl 点击及第三方插件插入的物品链接，防止物品名写入搜索框；弹出可复制的带引号物品名浮动框，支持最近历史（默认20条）、多收藏列表、拖拽保存位置及一键加入/管理收藏，简化拍卖行物品名复制与管理。

## 安装
- 将本文件夹放入游戏 `Interface/AddOns/` 目录，例如：
  `Interface/AddOns/ItemNameCopier`
- 登录游戏并在插件管理中启用 `ItemNameCopier`。

## 使用
- 在任意聊天、背包或插件界面按住 `Shift` 或 `Ctrl` 并点击物品链接，插件会弹出可复制的带引号物品名浮动框，且不会把物品名写入拍卖行搜索框。
- 浮动框支持历史记录、收藏列表及拖拽移动；在拍卖行打开时会自动拦截写入行为。

## 保存数据
- 保存的变量名：`ItemNameCopierDB`，包含历史（history）、收藏（favorites）和位置数据。
